# 前端面试题库（双 JD）

基于 BOSS 直聘两份 JD 整理的面试题网站，**岗位切换互不干扰**，每题含参考解析与答题要点。

| 岗位 | 题量 | 侧重 |
|------|------|------|
| **H5 高级开发工程师** | 32 题 | 海外 H5、Hybrid、RTC、弱网、架构 |
| **前端开发工程师** | 26 题 | JS/HTML/CSS、Vue/React、jQuery/Bootstrap、Webpack、社交直播 |

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 http://localhost:5173

## 构建

```bash
npm run build
npm run preview
```

顶部 Tab 可切换 JD，选择会保存在浏览器本地。
